# Component Library Specification

## Foundations

1. App shell (header, role switcher, theme switch)
2. Sidebar navigation
3. Bottom navigation bar (mobile)
4. Floating emergency action button

## Data Display

1. Metric card
2. Section card
3. Timeline list
4. Tag/chip
5. Empty state panel
6. Bar chart block

## Forms and Input

1. Auth form fields
2. OTP row input
3. Upload dropzone
4. Search/filter controls
5. Schedule slot controls

## Consultation Components

1. Chat pane
2. Video pane
3. Prescription tile
4. Follow-up request card

## Billing Components

1. Payment summary card
2. Wallet card
3. Subscription plan card
4. Invoice history row

## Admin Components

1. Verification queue list
2. Fraud alert list
3. Audit event row
4. CMS action card

## Accessibility

- Interactive controls minimum 44px tap area.
- Keyboard focus visible and consistent.
- Semantic labels on inputs/selectors.
- Color usage not the sole indicator for status.
