# Design System - Second Opinion Doctor

## Brand Direction

Logo-driven palette inspired by provided mark:

- **Primary Cyan**: `#09ABC0`
- **Primary Green**: `#83C847`
- **Navy Text**: `#0F2F46`
- **Soft Blue Surface**: `#DFF7FB`

## Visual Principles

1. Clinical trust with calm gradients
2. Premium cards with subtle shadows
3. Rounded corners (12px to 24px)
4. Spacious layout with minimal clutter
5. Accessibility-first contrast and typography

## Tokens

### Color Tokens

- `--brand-cyan`
- `--brand-green`
- `--surface` / `--surface-soft`
- `--text` / `--text-soft`
- `--border`

### Spacing Tokens

- 4, 8, 12, 16, 20, 24, 32

### Radius Tokens

- `radius-sm`: 12
- `radius-md`: 16
- `radius-lg`: 20
- `radius-xl`: 24

### Typography Scale

- Display: 32-40
- Heading: 24-30
- Section Title: 18-22
- Body: 14-16
- Caption: 12-13

## Components by Density

- **Mobile first**: compressed vertical rhythm, sticky bottom nav.
- **Tablet/Desktop**: expanded two-column dashboard with sidebar.

## Motion

- 180-220ms transitions for hover/tap states.
- Slide/fade entrance for new panels.
- Micro-interactions on cards, buttons, and nav.

## Dark Mode

- Inverse surfaces with preserved brand accents.
- Consistent semantic token mapping for role dashboards.
