# User Flows

## 1. Patient Journey

1. Open app -> Splash -> Onboarding -> Signup/Login.
2. Complete profile (age, gender, country, history, allergies, meds, blood group).
3. Upload records (PDF/image/MRI/lab/voice/camera scan).
4. AI processes records (summary, timeline, symptoms, translation).
5. Create second opinion request:
   - choose specialty
   - choose doctor
   - urgency
   - symptoms and questions
6. Pay using Stripe/Razorpay/wallet/subscription.
7. Receive doctor response + secure chat.
8. Join audio/video consultation if required.
9. Download prescription and request follow-up.

## 2. Doctor Journey

1. Register and complete onboarding.
2. Upload KYC, license, degree certificates.
3. Admin verifies profile.
4. View incoming case queue with AI triage tags.
5. Accept/reject case, provide written opinion.
6. Schedule consultation slots and manage timezone availability.
7. Run consultation, share notes and prescriptions.
8. Track earnings, ratings, and case analytics.

## 3. Admin Journey

1. Login to admin control panel.
2. Monitor platform KPIs and fraud alerts.
3. Verify or suspend doctor accounts.
4. Manage disputes and support tickets.
5. Review payment/subscription health.
6. Audit user actions and PHI access.
7. Publish CMS content (blog/FAQ/legal/notifications).
