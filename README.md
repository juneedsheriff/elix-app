# Second Opinion Doctor

Premium cross-platform healthcare product blueprint with:

- Patient mobile-first super app UX
- Doctor operations workspace
- Admin enterprise control tower
- Security/compliance-aware architecture docs
- TypeScript component library and design tokens

## Quick Start

```bash
yarn install
yarn dev
```

Open `http://localhost:5173`.
